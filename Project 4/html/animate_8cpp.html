Detailed Description

Program to make an animation of the Ising system.

This program sets up a window and prints the Ising system. The program used is CImg, which is available from https://github.com/dtschump/CImg and http://cimg.eu/. This program is installed as "sudo apt install cimg-dev" on debian systems.

The system is set up randomly, and shows domains when enough cycles has been completed.
