Detailed Description

Holds the full model of an Ising system.

This class contains the lattice system and its own generator. This enables one Ising system to be completely independent of another one as the seeds are different

Finds the statistical quantities from the Ising model.

This function samples the model every tau times for Measurements. tau should be the thermalisation time

The variables E, sigmaE, M and sigmaM are modified to give the average per spin over these measurements. The acceptance_rate is the number of accepted spins divided by M

sigmaE = < E^2 > - < E >^2 sigmaM = < M^2 > - < M >^2
